## FindingSubsets
#
##### It prints the subsets of the given array

### INPUT : 
#### 1st Line : Size of Set
#### 2nd Ling : Array element separated by spaces
#
### Input :
#### 4
#### 1  2  3  4
### Ouput :

#### 1
#### 2
#### 1 2
#### 3
#### 1 3
#### 2 3
#### 1 2 3
#### 4
#### 1 4
#### 2 4
#### 1 2 4
#### 3 4
#### 1 3 4
#### 2 3 4
#### 1 2 3 4
#
#### Limitations :
#### It doesn't print in size order but appearance order.
